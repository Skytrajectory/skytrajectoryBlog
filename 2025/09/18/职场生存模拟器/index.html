🕹️ 游戏概述

《办公室生存模拟器》是一款基于Unity开发的文字冒险游戏。在每次游戏中，玩家将扮演一个随机生成的角色，尝试通过面试并度过三个月的试用期。角色的年龄、性别、外貌、种族等特质将影响剧情发展以及与NPC的互动，让玩家亲身体验现实生活中不同身份所受到的差异化对待。

🌟 核心特色

🎮 游戏机制

  • 随机角色生成： 姓名、性别、年龄、外貌、种族等属性会通过增益/减益效果影响游戏进程。
  • 多结局系统： 结局由玩家选择和角色特质共同决定——仅存在一个“真实结局”。
  • 对话树系统： 使用自定义JSON结构，支持：
    • 叙述、单人对话和双人对话
    • 限时选择和普通选择
    • 背景/角色/音频切换
  • 偏见模拟： 特定标签会触发独特的歧视事件，以模拟现实世界中的偏见。

⚙️ 系统功能

  • 保存/加载/删除： 完整恢复故事状态和角色数据（Web端不完全支持）。
  • 成就追踪： 记录已解锁的结局、极端好感度事件以及与标签相关的路径。
  • 设置菜单：
    • 全屏/窗口模式切换
    • 英文/中文语言切换（UI动态更新）
    • 音量控制（主音量、背景音乐、音效）
  • 对话日志： 回顾对话历史。
  • 可视化对话树编辑器： 仅在PC版本中可用。Web版本不支持对话树编辑。

📦 版本信息

平台 版本 支持的功能 限制
PC v1.0 ✅ 完整功能（保存/加载/设置/编辑器） ❌ 需要手动下载并解压
Web (H5) v1.0-H5 ✅ 核心对话 + 部分设置 ⚠️ 不支持完整存档、快捷键和对话树编辑

💡 注意： Web版本数据存储在浏览器缓存中，清除缓存会导致数据丢失。

🎮 操作控制

操作 输入方式
推进对话 鼠标点击 / 回车键 (Enter)
选择选项 鼠标点击
打开设置 点击侧边栏的设置按钮
返回主菜单 点击侧边栏的返回按钮
查看日志 点击侧边栏的日志按钮

🚀 如何启动

✅ PC版本

  1. 下载并解压游戏；
  2. 运行 OfficeSurvivalSimulator.exe；
  3. 选择语言和分辨率；
  4. 开始新游戏或加载已有存档。

如果从Unity编辑器启动，请务必将splashScreen设置为启动场景，否则游戏逻辑将无法正确初始化。

🌐 Web (H5) 版本

  1. 在支持WebGL的浏览器中打开游戏（推荐Chrome/Edge）；
  2. 等待加载完成；
  3. 开始游戏。

⚠️ Web版本在旧版浏览器或移动设备上可能会出现UI缩放问题。建议在桌面端全屏进行游戏。

💡 教育意义

这款严肃游戏旨在促进对职场多样性和包容性的同理心与认知。通过随机角色特质体验不同形式的偏见，鼓励玩家反思自身固有观念，并追求公平、客观和善良。

📁 资源与素材

  • 背景： 使用真实世界照片并添加滤镜和模糊效果
  • 立绘： 玩家头像是根据标签生成的；关键NPC拥有多种表情
  • 音频资源： 包含背景音乐（BGM）、角色情绪音效、UI音效、环境音
  • 对话引擎： 自定义的JSON驱动系统，支持分支节点逻辑
